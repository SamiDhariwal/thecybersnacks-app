# The Cyber Snacks

A clean Next.js starter for The Cyber Snacks.

## Stack

- Next.js with the App Router
- TypeScript
- Tailwind CSS
- ESLint

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

Open `http://localhost:3000` after starting the development server.

## Project Structure

- `src/app/page.tsx` is the home page.
- `src/app/layout.tsx` is the root layout and metadata.
- `src/app/globals.css` imports Tailwind and defines global styles.
- `public/` is for static files such as images and icons.
- `package.json` contains scripts and dependencies.

## Verification

```bash
npm run lint
npm run build
```

The dev server should show the starter page at `http://localhost:3000`.
